
### 1️⃣ What is the difference between var, let, and const?**
##### var
- Function-scoped, hoisted with `undefined`, can be reassigned and redeclared.
##### let
- Block-scoped, hoisted in temporal dead zone (TDZ), can be reassigned but not redeclared.
##### const
- Block-scoped, hoisted in TDZ, cannot be reassigned or redeclared, must initialize at declaration
### 2️⃣ What is the spread operator (...)?
- The spread operator is a syntax through which we can spread the data between arrays and objects. It came in ES6.

### 3️⃣ What is the difference between map(), filter(), and forEach()?
- Map is an array method that creates a function and applies it to an array, creating a new array but -leaving the previous array unchanged.
-Filter is an array method that filters or separates the elements in an array.
-For Each is an array method but it works almost like a loop in that it runs the function once on the -array.


### 4️⃣ What is an arrow function?
- The arrow function is a shorthand for writing functions. It came in ES6 and makes the code look more -professional.


### 5️⃣ What are template literals?
- Template literals, which we call template strings in simple language, here we use backticks instead of -normal quotations, using this we make our website dynamic.
